# 🌸 Bhoomi Rawat — Portfolio

A single-page personal portfolio website showcasing my projects, skills, and journey as a B.Sc. IT student passionate about web development.

## 🔗 Live Demo
[View Live Portfolio](https://bhoomirawat.github.io/bhoomirawat-portfolio/)

## 🛠️ Built With
- HTML5
- CSS3 (custom animations, blob effects, scroll reveal)
- Vanilla JavaScript

## Features
- Fully responsive single-page design with smooth scroll navigation
- Animated hero section with floating blob backgrounds and custom cursor
- Circular profile photo with hover effects
- About Me section with personal introduction
- Skills section categorized into Languages, Core IT, and Databases
- Projects showcase with hover animations
- Contact section with clickable Email, LinkedIn, GitHub, and Resume links
- Scroll-triggered reveal animations using Intersection Observer
